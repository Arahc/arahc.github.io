回忆录『暂别』

Arahc 11.0
  2023-05-16 08:00 2023-05-16 08:00 创建   2023-07-30 05:38:06 2023-07-30 05:38:06 更新  
  • 杂项

说道 AFO，或许还有人不知道这个缩写的全拼吧。不过，大家说是 AFO 表示 Away From OI，我倒不那么认为。或者说我不希望是这样。虽然 OI 的路程到这里确实要告一段落，但是未来还会有大学 CSP，有 ACM，信息学路上还有许多其他的风景，而作为高中生 OI 的结束只是信息学的一个暂别而已。

在高二和高三的时间里，博客（arahc.github.io）不再会以较高的频率更新，要更新肯定也是 OI 无关。

NOIP 的时候我就已经预料到可能继续省选也不会得到想要的结果，不过我却不甘愿明明拥有了省选的知识点，而止步在一场因开题错误挂掉的 NOIP 上。所以我仍然选择了继续，即使希望渺茫也没关系，一个人一生总会干些这种事情的。至于最后也没有进队，与其说让我失望，其实也是意料之中的事情。既然做出了选择，就要准备好承担不好的后果。

开始

OI 的开始来源于一次学校的选拔考试。回想起来还依稀能记得卷子上的一些题目的大概题意。那时候我没有接触过 OI，但是题目里面有勉强涉及经典的 OI 算法，一些人排队接水的贪心，之类。

迷迷糊糊地，考的也非常差，听说只有 30/100，然而又听说分数线只有二十来分，就来一中了。

我还很清楚的记得第一节信息课，大曹给我们抛出一个汉诺塔问题。询问我们三个圆盘需要多少步，很多人都答出来了七步；老师接着问如果是 个圆盘呢，答出 的人比上次少了很多；老师接着问如果有 根柱子呢，这次就没有人回答出来了。

第二节信息课开始真正上机写代码了，我们打开 dev-C++，老师要求我们把发下来的纸质材料上的文本一字不错地抄上去：

然后运行，电脑上弹出一个黑色的命令行，好像电影里黑客的电脑，不过上面的内容没有那么复杂。一个数字 出现在最开头。

后面陆续有很多课，从顺序结构到分支到循环，再到数组、函数……第一个接触的“算法”叫模拟。那个时候参加培训的人非常非常多，有好几个机房。后来几乎每次选拔考试都会少很多人，竞争也很激烈。

我还记得在学完函数的某次选拔考试里，第一题是一个很模板的递归题……

个汉堡包，每个汉堡包用一个字符串表示，一个字符表示汉堡的一层。

今天来了一个奇怪的客户，他点了一个巨无霸汉堡，这个汉堡的制作方式是：在第一个汉堡的每层中间放一个第二种汉堡，第二种汉堡的每层中间放一个第三种汉堡……需要从上到下输出巨无霸汉堡每一层的字符。

当时忘记递归的我还非常纠结与如何设计一个循环层数是输入的 的多层循环。现在看起来傻里傻气的。

后来不断的选拔，初三时最后人数终于一个机房能装下了。这一届的一中信息组就此成立了。

OI

当然最让我印象深刻的就是从初二开始就和（bei）我（wo）一（la）起（zou）去吃饭的 @XeniaF @zenghaowei @Ckger，那段时候很喜欢到对面的粉店，然后人手一碗牛肉汤粉，然后沉迷 phigros 呜呜。

基本上从刚入坑开始一直到 rks 15+ 都是大家一起见证的！想起 @zenghaowei 看我直播解痉挛，@XeniaF 在我的带（fan）动（du）下入坑了 phigors 然后萌萌地问我为什么所有的键都变红了 ww，还有 @Ckger 看我要二见杀歌的时候不停的膜膜膜……

初三的时候还是非常快乐的，几个人天天聚在一起，还换了洛谷头像成 mopemope 的几朵花，@XeniaF 是橙色的????，@zenghaowei 是蓝色的????，@Ckger 是紫色的????（原曲绘里只有一半所以还自己去画图拼接了一下 233），然后我没其他颜色的花就手动换了个美穗 qwq。

后来四个人就有点小团体了 XD 基本上每次换座位都会想着坐在一起。很多分组的任务都会想着一起。

在这个懵懵懂懂的时光里，写出了获赞数最高的题解——最大子段和 ，感谢每一个点赞评论的人，由于较多，不胜枚举。

@zenghaowei 的脸很软。

中考的时候和 @XeniaF 一个考场！姐姐好强贴贴姐姐。

不知不觉就高一了，高一给我的记忆是最模糊的。可能是最不愿意想起的吧。

csp 的时候全信息组集体挂掉，廊桥分配三分的三分没 sort 的没 sort 总之挂分方案有很多但是却是都挂了。但是 noip 没有和 csp 一样人均 100-，这是好的。

邻近省选的时候不停的考做死的考烦死了 :(

这段时间主修音游逐渐从 phigros 转移到 arcaea，后来 phi 就弃了（

进行了一个 apio 的参加！给我的很好的印象就是酒店的伙食超级好超级长一条的餐桌然后自助餐 w，每天上完课会房间就和沙皇一起快乐音游。zhs orz orz orz orz orz orz，人家游记博客里说我打的 10.8 现在已经变成 10.9 了（雾

apio 算是很快乐的一段日子，但是前后都是不太有意思的培训和考试。混着混着高一就这么过去了，也没有留下非常新鲜的事情，这段时间在我的博客里写了很多的笔记和题目的题解，是博客更新的最高峰。只是大部份的笔记没有放在洛谷里，而是我的 github 博客站 Arahc’s home 中。

和 @zenghaowei 和 @Ckger 和 @XeniaF 的小团体有一段时间不是很愉快，后来 @Ckger 就提前告别信息组了。本来换成 mopemope 头像的四人组陆续又把头像换成了自己习惯的头像。那段花田成为了过去。

一切的消逝都是在等待被注入新的活力获得新生，花田也不例外。

高二是一段很开心的时间呢，和大家猜测的高二压力最大之类的是不是完全不一样呀。对于我而言最难受的日子莫过于高一。

随着和花田的复合，虽然头像没有换回来，但是小团体还是那个小团体（雾。还能记得 @XeniaF 过生日的时候，我们发了一个这样的很要素和整活味的犇犇：

今天 XeniaF 日 s 超天酱生烈华，转发这条消息，你今日就能上 211，ptt 999999。是的，我是试了，但是国际大师 XF 是真的假。因为 Arcaea 的烈华 At 都过不了 XeniaF。我祝最可爱的绝快生！乐

还有三个人生日送礼物的互相攀比，一定要送给别人最好的礼物，写出最用心的祝福 ww

noip 之前的时候搞颓风气弥漫，人均进化大师，然而 noip 的时候所有玩进化的都挂了，问题不大。抱着翻省选（其实更多的是能和大家多在一起一点）的想法，和教练说了自己要留下来的愿望。

当然，@zenghaowei 和 @XeniaF 也留下来了！

虽然 noip 和省选那段时间每天又是没有什么意思的重复的培训，但是还是发生了许多很有意思的事情。比如……

  • 神秘的机房失踪案件！失踪 OIer 在小机房被发现，发现时衣冠不整，面色泛红，神情模糊，肌肉无力，在多方调查下，真相逐渐浮出水面！（PS：其实是 @Arahc@XeniaF@Yahbim@PrincessQi 组团去小机房睡觉）
  • 震惊！今年的团建地点居然不是烈士公园！感动信息组十万年！
  • 某大爷晚上全屏打红警突然被叫到办公室！背后的原因竟然是……
  • 具有典型东亚男人思维的，两倍 zky，脚踩 csy 的神仙 diwanul 一拳把地球打爆！
  • 51nod 官方爆喷信息组出的构造题 std 跑样例和样例输出不一样！
  • 机房出现了一对 win！
  • ……

这段时间是一个快乐的时间，在学 OI 之余，通常是一天的任务结束，或者用午休的时间，从写终端小游戏开始，到接触 QT，不过 QT 并没有认真学，再后来看着 Unity 入门教程自己做了一个上下各 4key 的上落+下落音游，并且手捏了半个谱子。后来还有兴趣入坑一下 MIDI 音乐制作，不过这个就和 OI 完全没关系了，也没什么时间零基础地学，希望大学有时间试一试。

心情很不好的时候就会找 @XeniaF 倒垃圾，感谢 @XeniaF 耐心安慰 qwq。

省选结束的时候大部分信息组的人都一起去了 KTV 嗨歌，晚上还组团看了《铃芽之旅》！这是我的第一次线下看新海城的电影，音乐给人的感觉超级棒！比在线看好到不知道哪里去了！

结语

如果真要说成绩的话，我也没有在 OI 上搞出什么成绩来，不过成绩什么的当然是过去了就过去了，过程才是最值得回忆的。能在信息组遇到大家成为朋友就是我最大的幸福了。如果再给我一次机会，我仍然会选择 OI，决不会后悔。

那么，文化课也要加油呢～

最后，祝愿我们都有光明的未来。
