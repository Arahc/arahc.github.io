第一类斯特林数

基础

定义：nnn 个数组成 kkk 个圆排列的方案数，为 n,kn,kn,k 的第一类斯特林数。记作 [nk]n\brack k[kn​]。

第一类斯特林数的递推式：

[nk]=[n−1k−1]+[n−1k](n−1){n\brack k} = {n-1\brack k-1} + {n-1\brack k}(n-1) [kn​]=[k−1n−1​]+[kn−1​](n−1)

意义为：第 nnn 个数可以单独形成一个圆排列，也可以选择前 n−1n-1n−1 个数的任意一个并放到它后面。

还有另一个递推式：

[nk]=∑i=1n[n−ik−1](n−1i−1)(i−1)!{n\brack k} = \sum_{i=1}^n {n-i\brack k-1}\binom{n-1}{i-1} (i-1)! [kn​]=i=1∑n​[k−1n−i​](i−1n−1​)(i−1)!

意义为：枚举 111 所在的圆排列大小 iii，剩下的数字组成圆排列方案数为 [n−1k−1]n-1\brack k-1[k−1n−1​]，选择除了 111 之外的其他数到 111 所在圆排列的方案数为 (n−1i−1)\binom{n-1}{i-1}(i−1n−1​)，这些数字组成圆排列的方案数为 (i−1)!(i-1)!(i−1)!。

当然有的时候也需要把斯特林数拆开，那么这个递推式是经常反过来用的。

根据斯特林数的定义，显然，同一行所有斯特林数的和为 n!n!n!。证明则考虑枚举长度为 nnn 的排列的循环数即可。

Luogu4609 建筑师

nnn 个楼排成一行，高度是 nnn 的排列。求有多少种方案满足从左看恰好能看到 aaa 个楼，从右看恰好能看到 bbb 个楼。

多组数据，T≤2×105,n≤5×104,a,b≤100T\leq 2\times10^5,n\leq 5\times10^4,a,b\leq 100T≤2×105,n≤5×104,a,b≤100。

题解

最高的建筑无论左右都一定能被看到，且挡住了后面所有的视线，以此为契机。

考虑把所有的建筑分为 a+b−1a+b-1a+b−1 个“建筑群”，每个建筑群中选出最高的建筑为代表，放在这个建筑群的最左边，挡住建筑群中其它的建筑。

如果大小为 kkk 的建筑群代表已经确定了，剩下的建筑怎么放无所谓。有 (k−1)!(k-1)!(k−1)! 种放的方案数值上等于一个圆排列。

除去最高的建筑后，从左边看有 a−1a-1a−1 个代表，从右有 b−1b-1b−1 个。一个代表为一个群。于是变成了将 n−1n-1n−1 个建筑划分为 a+b−2a+b-2a+b−2 个建筑群，然后选择其中 a−1a-1a−1 个放在左边：

ans=(a+b−2a−1)[n−1a+b−2]ans = \binom{a+b-2}{a-1} {n-1\brack a+b-2}ans=(a−1a+b−2​)[a+b−2n−1​]

CF960G Bandit Blues

求有多少种大小为 nnn 的排列，满足其恰好有 aaa 个前缀最大值和 bbb 个后缀最大值。

a,b,n≤105a,b,n\leq 10^5a,b,n≤105。对 998244353998244353998244353 取模。

题解

如果一个数比它左边一个数小，它就不会是前缀最大值。也就是说一个数是前缀最大值当且仅当它左边没有数比它大。

ans=(a+b−2a−1)[n−1a+b−2]ans = \binom{a+b-2}{a-1} {n-1\brack a+b-2}ans=(a−1a+b−2​)[a+b−2n−1​]

第二类斯特林数

基础

定义：把 nnn 个不同的球放到 mmm 个相同的盒子里，盒子不能为空的方案数。记为 {nm}{n\brace m}{mn​}。

根据定义有第二类斯特林数的递推式：

{nm}={n−1m−1}+m{n−1m}{n\brace m} = {n-1\brace m-1} + m{n-1\brace m} {mn​}={m−1n−1​}+m{mn−1​}

意义为：考虑第 nnn 个球怎么放，要么放在一个已有的盒子里，要么新开一个。

有通项公式：

{nm}=1m!∑i=0m(−1)i(mi)(m−i)n{n\brace m} = \frac{1}{m!} \sum_{i=0}^m (-1)^i\binom{m}{i} (m-i)^n {mn​}=m!1​i=0∑m​(−1)i(im​)(m−i)n

证明考虑容斥，枚举至少多少盒子为空，在 mmm 里选出 iii 个盒子为空，剩下的 nnn 个球随意分配。注意到这样计算会认为盒子之间是不同的，因此最后要除掉一个 m!m!m!。

xn=∑i=0n(xi){ni}i!=∑i=0x{ni}xi‾x^n = \sum_{i=0}^n \binom{x}{i} {n\brace i} i! = \sum_{i=0}^x {n\brace i} x^{\underline i} xn=i=0∑n​(ix​){in​}i!=i=0∑x​{in​}xi​

证明

第二个等式显然（把组合数拆开，然后把阶乘约掉），考虑第一个等式。求其组合意义：nnn 个不同的球放入 mmm 个不同的个盒子的方案数，看作枚举有多少个盒子非空，然后把 nnn 个球放入的方案数的和，因为第二类斯特林数的盒子是相同的，所以还要乘上 i!i!i!。

题解

直接考虑连通这个条件不太好作，考虑连通块个数。nnn 很小，直接枚举点的子集划分。将其加入到 mmm 个子集内，强制不同点集的点不连边，相同点集的随意连。设按这种方法划分的方案数为 fmf_mfm​，最后恰好形成 mmm 个连通块的方案数为 gmg_mgm​。则 fmf_mfm​ 相当于在 gk(k≥m)g_k(k\geq m)gk​(k≥m) 种选择 mmm 个集合合并：

整个连通图连通的方案数为 g1g_1g1​：

ans=∑i=1n(−1)i−1(i−1)!fians = \sum_{i=1}^n (-1)^{i-1} (i-1)! f_ians=i=1∑n​(−1)i−1(i−1)!fi​

nnn 很小，复杂度不成问题，考虑怎么求 fff。

仍然枚举子集划分，考虑一个图的选择情况（选或不选）对应了一个未知数（000 或 111）。那么对于每一条确定有或者没有的边，我们都可以列出一个异或方程，形成异或方程组。而对于一个未知数 xxx 如果 xxx 为主元，则必须在或不在子集里，若 xxx 为自由元，则在或不在都可以。
